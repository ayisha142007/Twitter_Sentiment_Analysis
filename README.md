# Twitter Sentiment Analysis

This project performs sentiment analysis on Twitter text data using Python and Machine Learning techniques.
The model classifies tweets into "positive", "negative", or "neutral" sentiments.

## Project Overview 

- Preprocesses Twitter text data 
- Converts text into numerical features using vectorization
- Trains a machine learning model for sentiment classification  
- Predicts sentiment for custom user input sentences

 ## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- Matplotlip / Seaborn

  ## Conclusion

  This project demonstrates a basic implementation of sentiment analysis on Twitter data and can be extended using advanced NLP and deep learning techniques.
